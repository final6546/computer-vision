# 2D变换编辑器

一个基于Python和Pygame的交互式2D图形变换编辑器。

## 功能特性

- ✅ 添加矩形和图像
- ✅ 交互式选择和拖动形状
- ✅ 2D变换操作：
  - 平移（X, Y方向）
  - 旋转（0-360度）
  - 缩放（X, Y方向独立缩放）
- ✅ 两种操作模式：
  - 鼠标拖动直接移动
  - Shift + 拖动旋转形状
- ✅ 滑块精确控制变换参数
- ✅ 网格背景辅助定位
- ✅ 删除和清空功能

## 安装依赖

```bash
pip install -r requirements.txt
```

或者直接安装Pygame：

```bash
pip install pygame
```

## 运行程序

```bash
python transform_editor.py
```

## 使用说明

### 基本操作

1. **添加形状**
   - 点击"添加矩形"按钮创建彩色矩形
   - 点击"添加图像"按钮创建渐变图像

2. **选择形状**
   - 点击画布上的形状进行选中
   - 选中的形状会显示红色边框和中心点

3. **移动形状**
   - 直接拖动选中的形状进行移动

4. **旋转形状**
   - 按住 Shift 键，然后拖动形状进行旋转

5. **精确调整**
   - 使用左侧滑块精确控制：
     - 位置 X：水平位置（-400 到 400）
     - 位置 Y：垂直位置（-400 到 400）
     - 旋转：旋转角度（0 到 360度）
     - 缩放 X：水平缩放（0.1 到 3倍）
     - 缩放 Y：垂直缩放（0.1 到 3倍）

6. **删除和清空**
   - 点击"删除选中"删除当前选中的形状
   - 点击"清空画布"删除所有形状

### 快捷键

- **Shift + 拖动**：旋转模式

## 技术实现

- **语言**：Python 3
- **图形库**：Pygame
- **核心功能**：
  - 矩阵变换（平移、旋转、缩放）
  - 碰撞检测（点在旋转矩形内的判断）
  - 事件处理（鼠标交互）
  - UI组件（按钮、滑块）
